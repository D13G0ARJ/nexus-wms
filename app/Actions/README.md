# Actions Directory

This directory contains Action Classes following the Action Pattern.

Each action encapsulates a single business operation.

## Example Structure:
```
Actions/
├── Inventory/
│   ├── AdjustStockAction.php
│   ├── TransferStockAction.php
│   └── RecalculateStockAction.php
├── Products/
│   ├── CreateProductAction.php
│   └── ImportProductsAction.php
└── Orders/
    ├── CreateOrderAction.php
    └── ProcessOrderAction.php
```
